Task<IMobileDevice> Yandex.Detector.IMobileDetector.Detect ( IDictionary< string, object >  headers)

Performs request to Yandex.Detector web service and determines capabilities of mobile client device, using provided HTTP headers for its identification.

Parameters
headersMap of mobile device's HTTP headers with values.
Returns
Instance of IMobileDevice object, describing capabilities of identified mobile device.
Exceptions
ArgumentNullExceptionIf headers is a null reference.
DetectorExceptionIf there was error either during the request to Yandex.Detector web service, or mobile device cannot be identified based on a set of provided HTTP headers.
